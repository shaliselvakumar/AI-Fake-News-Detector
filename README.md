# AI Fake News Detector

This project is a Machine Learning model that classifies news articles as Real or Fake.

## Technologies Used
- Python
- NLP
- TF-IDF Vectorization
- Logistic Regression
- Scikit-learn
- Google Colab

## Model Performance
Achieved approximately 93% accuracy on test data.

## How It Works
1. Cleaned text (removed stopwords and punctuation)
2. Converted text into numerical features using TF-IDF
3. Trained Logistic Regression model
4. Evaluated using confusion matrix and classification report
